# راهنمای نصب بیواز روی سرور
### Ubuntu 24.04 — از طریق پروکسی گوشی — با Coolify

---

## دو روش نصب موجود است:

| روش | پیش‌نیاز | مناسب برای |
|-----|----------|------------|
| **روش اول: پروکسی گوشی** ✅ (توصیه شده) | گوشی Android با Termux + اینترنت | وقتی سرور اینترنت ندارد ولی گوشی دارد |
| **روش دوم: بسته آفلاین** | Docker Desktop روی ویندوز | وقتی هیچ اینترنتی در دسترس نیست |

---

# ─── روش اول: پروکسی گوشی (Termux) ─────────────────────────────────────────

## ایده کلی

```
گوشی Android (اینترنت دارد)
        │
        │  tinyproxy روی پورت 8888
        │
        └─── SSH Reverse Tunnel ──► سرور Ubuntu
                                    localhost:8888 = اینترنت گوشی
```

گوشی به عنوان پروکسی اینترنت برای سرور عمل می‌کند.

---

## قدم ۱ — روی گوشی (Termux): نصب ابزارها

```bash
# باز کردن Termux روی گوشی
pkg update -y
pkg install openssh tinyproxy -y
```

---

## قدم ۲ — روی گوشی: تنظیم tinyproxy

```bash
# ساخت فایل تنظیمات
cat > ~/tinyproxy.conf << 'EOF'
Port 8888
Listen 127.0.0.1
Timeout 600
Allow 127.0.0.1
EOF

echo "فایل تنظیمات ساخته شد."
```

---

## قدم ۳ — روی گوشی: راه‌اندازی پروکسی

```bash
# اجرای tinyproxy
tinyproxy -c ~/tinyproxy.conf

# بررسی اجرا
sleep 2
pgrep tinyproxy && echo "پروکسی در حال اجراست." || echo "خطا در راه‌اندازی پروکسی!"
```

---

## قدم ۴ — روی گوشی: اتصال SSH با تونل معکوس

```bash
# IP سرور را وارد کن (مثلاً 185.x.x.x)
SERVER_IP="IP_سرور_را_اینجا_بگذار"

ssh -R 8888:localhost:8888 root@$SERVER_IP
```

این دستور:
- به سرور SSH می‌زند
- پورت ۸۸۸۸ سرور را به پورت ۸۸۸۸ گوشی وصل می‌کند
- یعنی سرور می‌تواند از `localhost:8888` به اینترنت گوشی دسترسی داشته باشد

> **نکته:** این پنجره SSH باید باز بماند تا نصب کامل شود!

---

## قدم ۵ — روی سرور: کلون کردن پروژه

در پنجره جدید ترمینال، به سرور SSH بزن:

```bash
ssh root@IP_سرور
```

سپس روی سرور:

```bash
# نصب git اگر نیست
apt-get install -y git 2>/dev/null || true

# کلون پروژه
git clone https://github.com/USER/beewaz-web.git /root/beewaz
cd /root/beewaz
```

یا اگر نمی‌خواهی git کلون کنی، فقط اسکریپت را دستی بساز:

```bash
mkdir -p /root/beewaz/scripts
# محتوای scripts/install-via-proxy.sh را paste کن
nano /root/beewaz/scripts/install-via-proxy.sh
chmod +x /root/beewaz/scripts/install-via-proxy.sh
```

---

## قدم ۶ — روی سرور: اجرای اسکریپت نصب

```bash
cd /root/beewaz
bash scripts/install-via-proxy.sh
```

این اسکریپت به ترتیب:
1. بررسی می‌کند پروکسی در دسترس است
2. Docker را نصب می‌کند
3. Docker را برای استفاده از پروکسی تنظیم می‌کند
4. Coolify را از اسکریپت رسمی نصب می‌کند
5. آدرس پنل Coolify را نشان می‌دهد

**مدت زمان:** ۱۰ تا ۲۰ دقیقه

---

## قدم ۷ — تنظیم اولیه Coolify

بعد از نصب، مرورگر را باز کن:

```
http://IP_سرور:8000
```

### ۷.۱ — ساخت حساب ادمین
اولین بار صفحه ثبت‌نام نشان می‌دهد. ایمیل و رمز بگذار.

### ۷.۲ — تنظیم Server
منوی چپ → **Servers** → **Add Server**

- Server IP: `localhost`
- Private Key: **Generate a new key** → کلید عمومی را کپی کن

```bash
# روی سرور:
nano ~/.ssh/authorized_keys
# کلید عمومی را paste کن
```

سپس **Validate Server** را بزن.

---

## قدم ۸ — اضافه کردن Beewaz به Coolify

### ۸.۱ — ساخت Project
**Projects** → **New Project** → نام: `beewaz`

### ۸.۲ — اضافه کردن Application
**New Resource** → **Docker Compose**

محتوای `docker-compose.yml` پروژه را paste کن.

### ۸.۳ — تنظیم Environment Variables

```
POSTGRES_PASSWORD    = یک_رمز_قوی
AUTH_SECRET          = (openssl rand -base64 32)
NEXTAUTH_URL         = http://IP_سرور
NEXT_PUBLIC_APP_URL  = http://IP_سرور
ADMIN_PHONE          = 09120000000
ADMIN_PASSWORD       = رمز_ادمین
```

### ۸.۴ — Deploy
دکمه **Deploy** را بزن.

> این روش نیاز به Docker Desktop روی ویندوز دارد.

## نقشه کلی

```
کامپیوتر ویندوز (اینترنت دارد)          سرور Ubuntu 24.04 (بدون اینترنت)
─────────────────────────────           ──────────────────────────────────
 قدم ۱: نصب Docker Desktop
 قدم ۲: اجرای make-bundle-full.ps1
         ├─ دانلود پکیج‌های Docker      ──USB یا شبکه──►  قدم ۳: نصب Docker
         ├─ Build ایمیج Beewaz          ──  داخلی   ──►  قدم ۴: نصب Coolify
         ├─ Pull ایمیج‌های Coolify       ──────────────►  قدم ۵: راه‌اندازی Beewaz
         └─ beewaz-full-bundle.zip
```

**حجم تخمینی بسته نهایی: ۳ تا ۵ گیگابایت**

## قدم ۲ — ساخت بسته کامل آفلاین


| مرحله | کار |
|-------|-----|

# ─── عیب‌یابی ──────────────────────────────────────────────────────────────

**پروکسی گوشی کار نمی‌کند:**
```bash
# روی سرور تست کن:
curl -sf -x http://localhost:8888 https://google.com -o /dev/null && echo OK || echo FAIL
# اگر FAIL: مطمئن شو تونل SSH روی گوشی فعال است
```

**Coolify باز نمی‌شود:**
```bash
cd /data/coolify/source
docker compose logs coolify --tail=50
docker compose ps
```

**Beewaz باز نمی‌شود:**
```bash
docker logs beewaz-app --tail=50
docker logs beewaz-nginx --tail=20
```

**پورت ۸۰۰۰ بسته است:**
```bash
ufw allow 80 && ufw allow 443 && ufw allow 8000 && ufw allow 22 && ufw enable
```
