## قدم ۳ — استخراج بسته

### اگر از USB منتقل کردی:
```bash
# پیدا کردن USB
lsblk
# Mount کردن (معمولاً sdb1)
mount /dev/sdb1 /mnt
cp /mnt/beewaz-full-bundle.zip /root/
umount /mnt
```

### استخراج:
```bash
cd /root
apt install -y unzip 2>/dev/null || true
unzip beewaz-full-bundle.zip -d beewaz-bundle
cd beewaz-bundle
ls -la
```

خروجی باید این باشد:
```
docker-packages/     ← پکیج‌های Docker .deb
images/
  beewaz/            ← app.tar، postgres16.tar، nginx.tar
  coolify/           ← coolify.tar، traefik.tar، ...
coolify/             ← docker-compose.yml
nginx/               ← default.conf، certs/
scripts/             ← اسکریپت‌های نصب
docker-compose.yml   ← برای Beewaz
.env.example
```

## قدم ۵ — نصب Coolify

این اسکریپت:


## قدم ۶ — تنظیم اولیه Coolify

مرورگر را باز کن و به `http://IP_سرور:8000` برو.

### ۶.۱ — ساخت حساب ادمین
اولین بار صفحه ثبت‌نام نشان می‌دهد. ایمیل و رمز بگذار.

### ۶.۲ — تنظیم Server
در منوی سمت چپ: **Servers** → **Add Server**

چون سرور خود Coolify همان سرور است:
- Server IP: `localhost` یا `127.0.0.1`
- Private Key: **Generate a new key** را بزن → کلید را کپی کن

```bash
# روی سرور، کلید عمومی را اضافه کن:
nano ~/.ssh/authorized_keys
# خط کلید عمومی را paste کن
```

سپس **Validate Server** را بزن.

---

## قدم ۷ — اضافه کردن Beewaz به Coolify

### ۷.۱ — ساخت Project
**Projects** → **New Project** → نام: `beewaz`

### ۷.۲ — اضافه کردن Application
**New Resource** → **Docker Compose**

در باکس Docker Compose، محتوای `docker-compose.yml` بیواز را paste کن.

### ۷.۳ — تنظیم Environment Variables
در تب **Environment Variables** موارد زیر را وارد کن:

```
POSTGRES_PASSWORD    = یک_رمز_قوی
AUTH_SECRET          = (openssl rand -base64 32)
NEXTAUTH_URL         = http://IP_سرور
NEXT_PUBLIC_APP_URL  = http://IP_سرور
ADMIN_PHONE          = 09120000000
ADMIN_PASSWORD       = رمز_ادمین
```

### ۷.۴ — Deploy
دکمه **Deploy** را بزن.

چون ایمیج‌ها از قبل روی سرور لود شده‌اند، نیازی به pull از اینترنت ندارد.

## قدم ۸ — تنظیم Domain در Coolify (اختیاری)

اگر دامنه داری و DNS آن روی این سرور ست شده:

در Coolify → Application → **Domains** تب:
```
beewaz.ir
www.beewaz.ir
```

Coolify به صورت خودکار SSL از Let's Encrypt می‌گیرد — **اما چون اینترنت نداری، self-signed استفاده کن.**

## عیب‌یابی

**Coolify باز نمی‌شود:**
```bash
cd /data/coolify/source
docker compose logs coolify --tail=50
docker compose ps
```

**Beewaz باز نمی‌شود:**
```bash
docker logs beewaz-app --tail=50
docker logs beewaz-nginx --tail=20
```

**ایمیج‌ها پیدا نمی‌شوند:**
```bash
docker images | grep -E "beewaz|coolify|postgres|nginx|traefik"
# اگر خالی بود، دوباره load کن:
docker load -i images/coolify/coolify.tar
```

**پورت ۸۰۰۰ بسته است:**
```bash
# بررسی فایروال
ufw status
# باز کردن پورت‌ها
ufw allow 80
ufw allow 443
ufw allow 8000
ufw allow 22
ufw enable
```

**فضای دیسک پر شد:**
```bash
# پاکسازی ایمیج‌های استفاده نشده
docker image prune -f
# پاکسازی کل (با احتیاط)
docker system prune -f
```

## نکات مهم امنیتی

- رمزهای `.env` را قوی انتخاب کن
- پورت ۵۴۳۲ (PostgreSQL) را از خارج ببند

- رمز Coolify Admin را پیچیده انتخاب کن
- بعد از راه‌اندازی، پورت ۸۰۰۰ را با Nginx یا Traefik پشت HTTPS بگذار
